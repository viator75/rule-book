前端開發資料夾命名規則：

專案名稱：
Views
Images
sass
stylesheets


|    資料夾    |                                     | 
|:-----------:|:------------------------------------|
| Views       | 放置html.erb畫面                     |
| Images      | 放置圖片，檔名一律使用小寫。            |
| sass        | 放置sass與scss檔案，style一律在此編輯  |
| stylesheets | 放置style檔案，此檔案不編輯。          |
| js          | 放置js檔案                           |
| lib	      | 放置lib或套件檔案                     |
| videos      | 放置影片檔案                          |
